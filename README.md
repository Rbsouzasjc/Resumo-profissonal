# Portfólio — Robson Leandro de Souza

Site pessoal de portfólio profissional com foco em Logística & Operações.

## 📁 Estrutura dos arquivos

```
robson-portfolio/
├── index.html   → Estrutura da página
├── style.css    → Estilos e layout
├── main.js      → Animações e menu mobile
└── README.md    → Este arquivo
```

## 🚀 Como publicar no GitHub Pages

### 1. Crie um repositório no GitHub
- Acesse [github.com](https://github.com) e faça login
- Clique em **New repository**
- Nome sugerido: `portfolio` ou `robsonlsouza.github.io`
- Deixe como **Public**
- Clique em **Create repository**

### 2. Faça upload dos arquivos
- Na página do repositório, clique em **Add file → Upload files**
- Arraste os 4 arquivos (`index.html`, `style.css`, `main.js`, `README.md`)
- Clique em **Commit changes**

### 3. Ative o GitHub Pages
- Vá em **Settings** (aba do repositório)
- No menu lateral, clique em **Pages**
- Em **Source**, selecione: `Deploy from a branch`
- Em **Branch**, selecione: `main` → pasta `/ (root)`
- Clique em **Save**

### 4. Acesse seu site
Após 1–2 minutos, seu site estará disponível em:

```
https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/
```

Exemplo: `https://robsonlsouza.github.io/portfolio/`
